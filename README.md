# vithisa-49m-ggf

ระบบรวมข้อมูล 300 สาขา + 500 หน่วยงานร่วมโครงการ ผ่าน Google Form เข้า PostgreSQL

## ภาพรวมโครงการ

| รายการ | รายละเอียด |
|--------|-----------|
| ผู้กรอกข้อมูล | 300 สาขา |
| หน่วยงานภายใน | 300 แห่ง (สาขาละ 1 แห่ง) |
| หน่วยงานนอก (ร่วมโครงการ) | 500 แห่ง (เข้าร่วมผ่านสาขา) |
| รวมหน่วยงาน | 800 แห่ง |
| ระยะเวลา | 1 พ.ค. 2569 - 31 ก.ค. 2569 (92 วัน) |
| ความถี่ | 1 record / 1 หน่วยงาน / 1 วัน |
| ประมาณการ records | 73,600 records (800 × 92 วัน) |
